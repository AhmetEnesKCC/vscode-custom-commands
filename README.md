# Commands

| **COMMANDS** | **FUNCTION**                                                  |
| ------------ | ------------------------------------------------------------- |

# ABBREVIATIONS

| **ABBREVIATIONS** | **MEANING**                         |
| ----------------- | ----------------------------------- |
| $fn               | Current file Name With Extension    |
| $fnw              | Current file Name Without Extension |
| $fon              | Current folder Name                 |
| $fp               | Opened file's path                  |
| $fps              | Opened files's path inside quotes   |
